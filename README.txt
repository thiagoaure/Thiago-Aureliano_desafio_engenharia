### O algoritmo foi desenvolvido em JavaScript conforme o meu compreendimento com todas as funções solicitadas. Durante o processo inicializei o repositório com o Node e adicionei apenas uma dependência (readline-sync) para fazer a leitura de dados no console. 
Para iniciar o projeto, recomendo:

- Clonar o repositório do github:

$git clone https://github.com/thiagoaure/Thiago-Aureliano_desafio_engenharia

- Após a instalação, entrar na pasta local onde o repositório foi baixado e executar o comando para instalação das dependências:

$npm install 

- Logo depois, no terminal, entre na pasta que o projeto está, e execute:

$node index.js


O programa irá executar, e deverá aparecer o menu de opções para escolher o que deseja ver.
